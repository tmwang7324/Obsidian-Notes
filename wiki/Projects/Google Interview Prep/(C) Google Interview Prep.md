# Google Interview Prep

A focused, **topic-by-topic** campaign to land a **New Grad Software Engineer offer at Google** — and, in the same motion, build durable mastery of data structures, algorithms, and DP that outlasts the interview. Solo project: the interview loop is the forcing function; lasting skill is the co-equal prize. The new-grad loop is **coding + behavioral** (no heavy system-design round — that's L4+).

This is the wiki-level overview — read it first; drill into the project directory only for working detail it doesn't cover.
