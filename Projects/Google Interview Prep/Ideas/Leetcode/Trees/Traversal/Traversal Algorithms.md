# Tree Traversal Algorithms

Index of the four core tree-traversal patterns. Each is split into its own note.

- [[Preorder]] — Root → Left → Right. Copying trees, prefix expressions.
- [[Inorder]] — Left → Root → Right. BST gives ascending sorted order.
- [[Postorder]] — Left → Right → Root. Bottom-up tasks: deleting trees, directory sizes, reverse polish notation.
- [[Breadth First Search Level]] — Level-by-level, top to bottom with a queue.
