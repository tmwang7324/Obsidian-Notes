# Trees
The condition or value to be recorded at a particular node (such as depth or similarity) should be decided before any iteration happens, as what occurs in [[Preorder|Preorder Traversal]].
However, the values from deeper nodes should propagate up in the graph or tree, and alter the current value of the node after all traversal has finished, as what occurs in [[Postorder|Postorder Traversal]].
