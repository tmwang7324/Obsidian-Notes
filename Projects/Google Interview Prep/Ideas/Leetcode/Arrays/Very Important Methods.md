`list.extend(list2)`: Takes all elements within list 2

| Methods               | Description                                                                                                     | Time Complexity |
| --------------------- | --------------------------------------------------------------------------------------------------------------- | --------------- |
| `list.extend(list2)`: | Takes all elements within list2 and appends them individually to list.<br>Can take any iterable as a parameter. | O(n)            |
| `.append()`           | Adds an element to the end of a list (can take lists as parameters)                                             | O(1)            |
| `.clear()`            | Removes all the elements from a list                                                                            | O(n)            |
|                       |                                                                                                                 |                 |
