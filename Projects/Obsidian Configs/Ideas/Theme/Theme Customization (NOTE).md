### Custom Background (NOTE) 
1. Go to the AnuPpuccin github and search for the custom background css.
2. Copy the CSS code into *.obsidian/snippet* folders. Enable the custom CSS snippet.
3. In Style Settings, there should be a section called "AnuPpuccin Custom Background." Drop it down and toggle "Enable Custom Background (WIP)." Ensure that "Colorful Frame" is toggled off.
4. Provide the link to the desired background image created using catbox.moe.
5. Some of the background picture urls I like: 
	1. https://files.catbox.moe/4hvv4y.jpg => Sunset City
	2. https://files.catbox.moe/mdni6i.jpg => Shelter by Porter Robinson (A lil bit too bright)
	3. https://files.catbox.moe/0tj4b8.jpg => Biking in the dusk
	4. https://files.catbox.moe/j5izf1.jpg => Starlight Pond
	5. https://files.catbox.moe/cgb1sq.webp => FRIEREN (Using Now)
6. Adjust the brightness, blur of the background image, as well as the opacity of the container.
7. To make the sidebar transparent, go to the "Workplace" section in the main "AnuPpuccin section" in Style Settings.
8. Go to the "Workplace Layout" section, and change the layout style to either "Border" or "Cards"
9. **Next Steps:** Find high quality background images to cycle around.


### Widening Note Interface (NOTE)
I was able to do this simply by toggling off the "Readable Line Length" option under *Display* in *Editor.* This enables the notes to stretch all the way from the left sidebar to the right sidebar.
