Daily notes should have a YAML frontmatter that contains date, tags (what is covered during the day), arrows that link to the next and previous day, productivity, as well as mood.

**Rationale:** My daily notes so far are a bit isolated to the rest of my content. Aside from its narrative input to *Progress,* daily notes do not connect to many notes pages, and are a bit difficult to navigate (hence the arrows).

I also want the current day's daily plan file to be embedded, but editable.

**Rationale:** Going back and forth between daily plans and the daily journal can get frustrating, so linking the two together solves that.



# Setting a Template for Daily Notes
1. Create a new note named "Daily Note Template" or anything I want.
2. Write what I want to keep in every note, such as date, agenda, pointers to yesterday and the next day, etc.
## Example: 
```md
# {{date:YYYY-MM-DD}}

## Tasks

- [ ]
```

3. Open **Settings**.
4. In the sidebar, click **Daily notes** under **Plugin Options**
5. In the text box next to **Template file location,** select the "Daily Template" note.
6. Finished!
