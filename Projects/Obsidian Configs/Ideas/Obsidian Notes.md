>**Notes for Obsidian Config**

**Backlog:** A backlog is a pile of "things to do next" for a project - concretely, the type: next-step files sitting in that project's Iteration Logs/ folder.


**Derived Task**: A derived task is a plan item the skill generates


## Themes
Some interesting themes I and the collective consensus have found interesting are AnuPpucin, Minimal, and Obsidianite (for amazing code color scheme and nice horizontal rules).

Almost all of the individual element features Obsidian supports are customized using the **Styles Settings** plugin or in **Appearances**.
