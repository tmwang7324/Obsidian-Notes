
# Pre-Signed URLs (Server-to-Client Approach)
A pre-signed URL is a time-limited, cryptographic link generated by my backend server. It gives a client temporary permission to upload or download a specific file directly to the cloud bucket without needing Firebase infrastructure.

**IMPORTANT:** File uploads orchestrated by a pre-signed URL bypass Firebase security rules, so I have to perform necessary validation checks before the URL is given.
